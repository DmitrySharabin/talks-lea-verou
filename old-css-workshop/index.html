IE Matrix filter

For transforms in IE < 9

/* rotate(45deg) */
filter: progid:DXImageTransform.Microsoft.Matrix(
  M11=0.70710678, M12=-0.70710678, 
  M21=0.70710678, M22=0.70710678,
  sizingMethod='auto expand');
-moz-transform:  matrix(.70710678,.70710678,-.70710678,.70710678,0,0);
-ms-transform:  matrix(.70710678,.70710678,-.70710678,.70710678, 0,0);
-webkit-transform:  matrix(.70710678,.70710678,-.70710678,.70710678,0,0);
-o-transform:  matrix(.70710678,.70710678,-.70710678,.70710678,0,0);
transform:  matrix(.70710678,.70710678,-.70710678,.70710678,0,0);

Finding parameters:

  • Apply transforms as usual (e.g. rotate(45deg))
  • Run in the console:
    getComputedStyle(element).WebkitTransform;
  • There you get your matrix parameters!
  • More robust: useragentman.com/IETransformsTranslator/
