Even More CSS Secrets

  • With multiply blend mode, black is cutout and white is opaque

Fancy borders

Variable Fonts for non-variable strokes

  • The weight axis in variable fonts allows us to equalize strokes across different font sizes
  • We can use linear or quadratic regression to avoid calculating the weight every time
